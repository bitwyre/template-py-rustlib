
## Pre-requisites

- Linux
- Nightly Rust
- Python 3.7 or later
- [cbindgen](https://github.com/eqrion/cbindgen) (for C/C++ not yet implemented)

## Installing the package

### Debug Build & Install (Python Only)

```bash
./rebuild-install.sh debug
```

### Release Build & Install (Python Only)

```bash
./rebuild install
```
